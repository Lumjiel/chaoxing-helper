# 超星学习通助手 (chaoxing-helper)

> "当你发现课程要求发 100 条讨论，而你已经不想活了的时候..."
>
> 这就是你需要的工具！🎉

一套 Tampermonkey 用户脚本，用于增强超星学习通的使用体验。让你的大学生活少一点重复劳动，多一点摸鱼时间。

---

## 📦 脚本列表

### 1. 学习通代码编辑器增强

解决超星学习通代码编辑器无法粘贴、格式错乱、复制受限问题，适配 Python/Java/C++ 等所有语言。

**功能特点：**
- 解除粘贴限制（妈妈再也不用担心我贴不上代码了）
- 解除复制限制
- 解除文本选择限制
- 修复 CodeMirror 编辑器粘贴格式错乱
- 支持页面动态加载内容的自动适配

---

### 2. 学习通智能发帖助手 ⭐

> 适用场景：课程要求发满 **100 条讨论**、发满 **100 条回复** 等让人怀疑人生的任务

一键解放双手的自动发帖工具，支持任务队列和断点续传。你只需要准备好内容，剩下的交给脚本。

**功能特点：**
- 📋 多内容队列管理（一次准备多条，轮流发布）
- ⏱️ 间隔时间可自定义（建议 3-5 秒，太快容易被判作弊）
- 💾 刷新页面自动续传未完成任务（浏览器崩了？不怕！）
- 🔄 失败自动回滚重试（网络波动？不存在的）
- 🎨 美观的操作界面（比学习通原生的好看一万倍）

## 🔧 安装使用

### 前置要求

1. 安装 **[Tampermonkey](https://www.tampermonkey.net/)** 浏览器扩展
   - 支持 Chrome、Firefox、Edge、Safari、Opera 等主流浏览器
   - 手机浏览器也支持（需要安装支持 Tampermonkey 的浏览器，如 Kiwi Browser）

### 安装脚本

**方式一：一键安装（推荐）**

直接点击下面的链接，在弹出窗口中点击"安装"：

- [安装代码增强器](dist/code-enhancer.user.js)
- [安装智能发帖助手](dist/auto-poster.user.js)

**方式二：手动安装**

1. 打开 Tampermonkey 扩展图标
2. 点击"+"号创建新脚本
3. 清空编辑器内容
4. 复制 `src/auto-poster/index.js` 的全部内容粘贴进去
5. 保存（Ctrl+S）

## 🚀 智能发帖助手 - 详细使用指南

### 适用课程页面地址

```
https://mooc1.chaoxing.com/activity/activsyllabus/student/index
```

或者进入课程 → **讨论区** → 任意一个讨论话题页面

### 使用步骤（包学包会）

**第一步：进入讨论区页面**

进入课程后，找到"讨论"或"论坛"入口，进入任意讨论页面。

**第二步：你会看到两个发帖框（重点！！！）**

😅 没错，学习通就是这么离谱。一个页面给你塞两个编辑器：

```
┌─────────────────────────────────────────┐
│  📝 发表话题（可折叠）❌ 这个是假的 ❌    │  ← 点 ▲ 折叠它
├─────────────────────────────────────────┤
│  📝 发表话题（可折叠）✅ 这个才是真的 ✅  │  ← 用这个！
└─────────────────────────────────────────┘
```

⚠️ **重要**：上面那个框是假的！是学习通 UI 的 bug！发布内容后你会发现它根本没发出去！

脚本已经帮你处理好了，**只需要关注下面的编辑器**。

**第三步：打开发帖助手面板**

安装脚本后，页面右上角会出现一个紫色面板：

```
┌─────────────────────────────┐
│ 📝 学习通发帖助手 v12.0    │
│ 剩余: 5                     │
│ ▲                           │
├─────────────────────────────┤
│ 发帖标题：[___________]      │
│ 内容列表：                  │
│ [_____________________]     │
│ [_____________________]     │
│ [_____________________]     │
│ 间隔(秒)：[3]               │
│ ☐ 刷新自动续传              │
│                             │
│ [🚀 开始任务] [⏹ 结束]     │
│ 状态：等待开始...           │
└─────────────────────────────┘
```

**第四步：配置你的任务**

1. **发帖标题**：每条讨论的标题（可以都一样，比如"打卡"）
2. **内容列表**：这里写你要发的内容，**每行一条**：

   ```
   第一条讨论的内容
   第二条讨论的内容
   第三条讨论的内容
   ...（想发多少写多少）
   ```

3. **间隔时间**：建议 **3-5 秒**（发太快可能被学习通检测到异常）
4. **刷新自动续传**：勾选后，刷新页面会继续未完成的任务

**第五步：点开始，然后摸鱼去吧 🦥**

点击"🚀 开始任务"，脚本会自动：
1. 点击下面的真编辑器
2. 填写标题和内容
3. 点击发布
4. 等待间隔时间
5. 重复以上步骤直到发完所有内容

## ❓ 常见问题与解决方案

### Q1: 点了开始但没反应？

**A：** 可能是以下原因：

1. **没进入正确的页面** - 必须进入讨论区页面，不能是课程首页
2. **脚本没安装成功** - 检查 Tampermonkey 扩展是否启用，脚本是否显示"已启用"
3. **页面还没加载完** - 等页面完全加载后再点开始

### Q2: 发了几条就停了？

**A：** 正常情况！可能是：

1. **网络波动** - 脚本会自动等待 3 秒重试
2. **学习通抽风** - 等待时间可能会自动延长
3. **被检测为机器人** - 建议把间隔调到 5 秒以上

### Q3: 两个编辑器分不清哪个是真的？

**A：** 小技巧：
- 随便在下面的编辑器输入文字，然后按 Ctrl+Enter 发布
- 如果内容没出现在讨论区列表里，说明你点的是假的
- 脚本会**自动识别下面的真编辑器**，你不用管

### Q4: 刷新页面后任务不见了？

**A：** 确保勾选了"**刷新自动续传**"选项。没勾选的话，刷新=重新开始。

### Q5: 标题或内容发重了？

**A：** 这是学习通编辑器的缓存问题，不是脚本的锅。可以在内容里加序号：
```
讨论1
讨论2
讨论3
```
或者加随机字符。

### Q6: 想中途停止怎么办？

**A：** 点"⏹ 结束"按钮。如果想完全清空任务，点结束后确认即可。

### Q7: 页面报错了但脚本还在跑？

**A：** 脚本有容错机制，会自动重试。如果一直失败，建议：
1. 刷新页面
2. 取消勾选"刷新自动续传"
3. 手动检查问题

## ⚠️ 注意事项

1. **间隔时间别设太短** - 建议 3 秒以上，不然可能触发学习通的风控
2. **内容要有变化** - 连续发一模一样的帖子可能被老师发现（虽然老师一般不看）
3. **不要在深夜跑** - 服务器负载高，容易失败
4. **这是应急工具** - 有空还是自己发吧，毕竟是自己的学业
